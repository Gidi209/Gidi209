# Política de Privacidade

## Coleta de Dados

Esta seção descreve quais tipos de informações podem ser coletadas pelos projetos disponíveis neste perfil do GitHub.

| Tipo de Dado             | Coletado   | Descrição                                                          |
| ------------------------ | ---------- | ------------------------------------------------------------------ |
| Nome                     | Opcional   | Fornecido voluntariamente pelo usuário em formulários ou mensagens |
| Endereço de Email        | Opcional   | Utilizado para responder a contatos ou sugestões                   |
| Informações do Navegador | Automático | Dados técnicos como tipo de navegador ou sistema                   |
| Endereço IP              | Automático | Utilizado para segurança e funcionamento do serviço                |

---

## Uso das Informações

As informações coletadas podem ser utilizadas para os seguintes fins.

| Finalidade            | Descrição                                          |
| --------------------- | -------------------------------------------------- |
| Comunicação           | Responder mensagens, dúvidas ou sugestões          |
| Melhoria dos Projetos | Aprimorar funcionalidades e qualidade dos projetos |
| Segurança             | Prevenir abuso, spam ou atividades maliciosas      |

---

## Serviços de Terceiros

Alguns projetos podem utilizar serviços externos.

| Serviço   | Finalidade                            |
| --------- | ------------------------------------- |
| Netlify   | Hospedagem de sites                   |
| Formspree | Processamento de formulários          |
| GitHub    | Armazenamento de código e colaboração |

---

## Retenção de Dados

| Tipo de Dado       | Período de Armazenamento                      |
| ------------------ | --------------------------------------------- |
| Mensagens enviadas | Apenas pelo tempo necessário para responder   |
| Registros técnicos | Temporariamente para monitoramento do sistema |

---

## Direitos do Usuário

Os usuários podem solicitar:

| Direito  | Descrição                             |
| -------- | ------------------------------------- |
| Acesso   | Saber quais dados foram coletados     |
| Correção | Corrigir informações incorretas       |
| Exclusão | Solicitar a remoção de dados pessoais |

---

## Atualizações da Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças nos projetos ou serviços.

Caso tenha dúvidas sobre esta Política de Privacidade, abra uma *issue* no repositório ou entre em contato através do GitHub.
